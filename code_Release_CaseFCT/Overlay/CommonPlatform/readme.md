### MacOS Release
- run "createMacApp."
- 自动生成 "PRMTester.app" 在./dist/目录下
- 双击PRMTester.app即可启动平台

### Windows Release
- run "createWindowsApp.bat"
- 自动生成 "PRMTester" 工程文件夹在当前目录
- 双击PRMTester/PRMTester.exe即可启动平台

### Linux Release
- run "bash createLinuxApp.sh"
- 自动生成 "PRMTester" 文件夹在当前目录
- 移动 "PRMTester" 文件夹到工作路径
- 运行 "PRMTester" 文件夹里面的 "bash createDesktop.sh"
- 用户桌面生成 "PRMTester" 快捷链接，点击运行即可
